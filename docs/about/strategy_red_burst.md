# 🔥 紅爆策略

**「紅爆」= 創歷史高點 + 爆量** 的中長線跟隨策略,經過 **9 年全市場、9094 筆 D1 候選 / 3992 筆實際進場** 完整回測(拆股還原 K 線)。

> 跟黑飛舞的差別:**黑飛舞短線(平均 6 天)、紅爆中長線(平均 30 天)**。兩者互補,不是替代。

---

## 進場條件

### Step 1:**D1 加入追蹤**(觀察名單)

| # | 條件 | 直白解釋 |
|---|------|---------|
| 1 | **D1 創 400 日新高** | 突破近 1.5 年高點,代表結構性突破 |
| 2 | **D1 量 ≥ 20 日均量 3 倍** | 量大有共識,主力進場 |

當天滿足 → 進入「**紅爆預備軍**」。

### Step 2:**D5 進場驗證**(D4 / D5 都要強)

| # | 條件 |
|---|------|
| 3 | **D4 收盤 ≥ D1 收盤** |
| 4 | **D5 收盤 ≥ D1 收盤** |

兩個都滿足 → **D5 收盤(13:00 後)進場**。

D2 / D3 可以收黑、跌破 D1 close 都沒關係 — 只要 D4 / D5 都站回 D1 close 之上,代表「強勢回歸」。

歷史上 **D1 → D5 通過率 43.9%**(9094 D1 → 3992 進場)。

---

## 出場條件 — 純機械式 20 天結算

進場後不設停損停利,改成**滾動 20 天視窗結算**:

```
進場 → +20 天結算:
  過去 20 天漲幅 ≥ 5% → 續抱(下個 20 天再結算)
  過去 20 天漲幅 < 5% → 出場
```

**例**(進場 100 元):
- Day 20:漲到 108(+8% ≥ 5%)→ 續抱
- Day 40:從 Day 20 的 108 → 110(+1.85% < 5%)→ **出場**(總報酬 +10%)

機械式設計,不需要主觀判斷。

## 🤔 風險與限制

### 1. 單筆損失可能很大

最大輸單 **-95.6%**(世紀* 2025)。沒緊停損 = 黑天鵝可能讓部位歸零附近。

對策:
- **單筆部位 ≤ 5%** 總資金(最大虧也只蝕 5% × 95% = 4.75%)
- **絕不重壓單一紅爆標的**

### 2. D1 重複觸發

有些股(如 8021 尖點)連續多天創新高 + 爆量 → D1 連續觸發多次。回測對「重複觸發同一股」沒去重,實際策略需考慮。

### 3. 中位數仍是 -2.77%

代表「**多數單筆會小虧**」,期望值正是被少數飆股單(+300% 以上)拉起來。心理建設:**3 筆裡有 2 筆是輸的**。

### 4. 平均持有 30 天 → 資金週轉慢

跟黑飛舞 6 天的快進快出不同,紅爆**資金被綁住一個多月**。資金規模較小者要注意機會成本。

## 操作建議(尚待 forward tracking 驗證)

| 階段 | 做法 |
|---|---|
| 看到 D1 觸發 | 進「預備軍」名單,**不立即進場** |
| 等待 D2-D4 | 觀察是否續強(收回 D1 close 之上) |
| D5 收盤前 13:00 後 | D4 close ≥ D1 close 且 D5 收盤已≥ D1 close → **小資金試單(2-3% 總資金)** |
| 進場後 +20 天 | 結算,過去 20 天漲幅 ≥ 5% → 續抱 |
| 下個 20 天結算 | 同上規則 |
| 任一 20 天漲幅 < 5% | **出場**,釋放資金等下一筆 |
