# 🎯 黑飛舞策略

**「黑飛舞」= 創高雙黑站5MA(強勢回檔)**

這是一個**台股短線「強勢股回檔買點」進場策略**,經過 1 年全市場回測驗證。

---

## 進場條件(全部要符合)

| # | 條件 | 直白解釋 |
|---|------|---------|
| 1 | D-1 創 60 日新高 | 昨天創出近 3 個月新高 |
| 2 | D-1 收黑 | 昨天衝高被打回(疲弱訊號) |
| 3 | D 不創高 | 今天沒突破昨天的高點(動能繼續弱) |
| 4 | D 收盤站上 5MA | 但短線結構未破壞 |
| 5 | 5MA > 10MA, 20MA, 60MA, 120MA | 完整多頭排列(長期仍多頭) |

**流動性過濾**:近 20 日均量 ≥ 2000 張(避免小型股流動性陷阱)

---

## 為什麼這幾個條件加在一起有意義

- **創 60 日新高 + 收黑**:衝高被殺 = 短期賣壓出盡
- **D 不創高**:還沒有更多新買盤,空間清乾淨
- **站穩 5MA + 5MA 為最高均線**:**核心趨勢沒破** — 這只是「強勢股的健康回檔」,不是趨勢反轉

簡單說:**「強勢上漲股,衝高被打回後的低風險上車點」**

---

## 出場策略(四段式動態停利)

進場後依「進場後最高漲幅」(peak %) 分四階段:

| 階段 | 觸發條件 | 出場規則 |
|------|---------|---------|
| **Stage A** | peak < +5% | 收盤跌破停損價(兩日低 -5 檔) |
| **Stage B** | +5% ≤ peak < +10% | 收盤跌破 5MA → 出場 |
| **Stage C** | +10% ≤ peak < +15% | 跌破 5MA **或** 從高點回檔 7% → 出場 |
| **Stage D** | peak ≥ +15% | 從高點回檔 12% → 出場(不再看 5MA,讓飆股跑) |

### 設計邏輯

- **剛進場利潤薄** → 用緊停損保命(Stage A)
- **站穩 +5%** → 改用 5MA 移動停利,讓利潤奔跑(Stage B/C)
- **漲超過 +15%** → 忽略 5MA(因為 5MA 已遠),只看「**從高點回檔 12%**」鎖大利潤(Stage D)

## 出場原因分布(觀察出場特性)

```
Stage A 觸及停損(兩日低-5檔)    主要 loss 來源
Stage B 收盤跌破 5MA            次主要出場
Stage C 從高點回檔 7%            短線飆股鎖利
Stage C 收盤跌破 5MA            較少觸發
Stage D 從高點回檔 12%          飆股奔跑後出場
觀察期內未出場                   30 天 hold 到末仍上漲(飆股)
```

**「Stage D 從高點回檔 12%」是大贏家來源** — 對應吃到 +15% ~ +30% 利潤的飆股。

---

## 適用範圍

✅ **適合**:
- 中大型股(流動性 ≥ 2000 張/日)
- 結構性多頭趨勢中的回檔
- 中期持有(平均 4-5 天,最長 30 天)

❌ **不適合**:
- 小型股(流動性差,滑價大)
- 弱勢股 / 下跌趨勢中的反彈
- 純當沖(本策略最少持有 1 天)
- 大資金量(會吃到自己)

---

## ⚠️ 重要提醒

1. **回測 ≠ 實際操作** — 真實滑價、心理因素、執行紀律都會降低勝率
2. **樣本期間 = 過去 1 年** — 牛市環境下回測,熊市表現可能不同
3. **未扣手續費** — 實際扣 0.4%/筆 後,期望值約 +2.19%/筆
4. **Forward Tracking** 才是真正驗證 — 至少累積 30 筆已結案再評估策略可行性
