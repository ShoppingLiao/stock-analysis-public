### 指標 1 — 每日漲停股

- 🟡 黃燈:30-50 檔
- 🔴 紅燈:> 50 檔 **且** 集中度 < 30%(分散多產業 = 末升段特徵)
- 🚨 超紅:> 80 檔(極端狂熱)

### 指標 2 — 低價小型股(垃圾股池)

- **池條件**:股價 < 30 元 **且** 股本 < 30 億 **且** 過去 6 月漲幅 < 10%
- 🟡 黃燈:池內漲停 ≥ 5 檔
- 🔴 紅燈:池內漲停 ≥ 10 檔
- 🚨 超紅:池內成交金額占大盤 > 5%

### 指標 3 — 融資餘額變化

- 🟡 黃燈:單日 +100~150 億
- 🔴 紅燈:單日 +150 億 **或** 近 5 日累計 +500 億
- 🚨 超紅:融資餘額 / 上市市值 > 0.5%

### 指標 4 — 券商質借限縮

**監控範圍**:台灣前 10 大本土券商(元大 / 凱基 / 富邦 / 永豐金 / 國泰 / 群益金鼎 / 元富 / 華南永昌 / 統一 / 日盛)+ 主要外資(瑞銀 / 摩根大通 / 美林 / 高盛 / 野村 等)

- 🟡 黃燈:2 家以上券商收緊質押
- 🔴 紅燈:5 家以上券商收緊或暫停
- 🚨 超紅:紅燈 + 龍頭三家(元大 / 凱基 / 富邦)中至少 2 家 suspend / freeze

### 指標 6 — 法人避險訊號(防禦類 + 反向 ETF)

- 🟡 黃燈:反向 ETF 買超 ≥ 5M 股 **或** TOP 30 買超防禦類 ≥ 3 檔
- 🔴 紅燈:反向 ETF 買超 ≥ 20M 股 **或** TOP 30 買超防禦類 ≥ 5 檔
- 🚨 超紅:紅燈條件 **且** TOP 30 賣超含半導體/電子 ≥ 3 檔(法人賣科技買防禦)

**防禦類定義**:金融保險 / 通信網路 / 電信 / 食品工業 / 電器電纜 / 公用事業

### 指標 5 — IPO 蜜月行情

- 🟡 黃燈:過去 30 天新股**首日平均漲幅 > 50%**
- 🔴 紅燈:連續 5 漲停以上新股 **≥ 3 檔**
- 🚨 超紅:首日平均漲幅 > 100%

### 🎢 組合指數

**SSI 散戶情緒指數**(由指標 1+2+3+4 加權)

- 冷靜 < 30 / 正常 < 60 / **偏熱 < 80** / 極端狂熱 < 100 / >100 歷史性瘋狂

**BPI 泡沫程度指數**(融資/市值 + P/E 偏離 + IPO 蜜月 + 漲停離散度 + 券商質借)

- 健康 < 40 / **偏高 < 60** / 泡沫形成 < 80 / 泡沫末期 < 100 / >100 隨時破裂
