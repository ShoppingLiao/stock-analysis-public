## 🎯 核心方法論

### DDMD 框架(大大毛多)

從 4 個維度評分(各 5 分,總分 20):

| 維度 | 看什麼 |
|------|------|
| **大(產業)** | 產業 TAM、成長率、結構性紅利、競爭格局 |
| **大(市佔)** | 公司在產業中的位置、護城河、定價權 |
| **毛(毛利率)** | 絕對水準、趨勢、可持續性、上限 |
| **多(成長性)** | EPS 階梯、未來 3 年動能、量化可預測性 |

### 風格

- **理性、有觀點、不模糊** — 看好就直說,看空也直說
- **量化優先** — 每個論點都帶數字
- **格式即訊息** — 表格、emoji、粗體傳達結構
