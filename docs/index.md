## 📊 DDMD 個股分析

用 **「大大毛多」** 框架系統化拆解個股,完整 A→F 六章節報告:

- **A 公司總覽** → 公司是誰、賣什麼、賣給誰、性感題材
- **B 大大毛多評估** → 產業 / 市佔 / 毛利率 / 成長性 四維度評分
- **C 評分 + 三句話敘事** → 0-20 分總評 + 「為何贏 / 怎麼賺 / 怎麼輸」
- **D 股價情境分析** → 悲觀 / 中立 / 樂觀 / 極度樂觀 四情境目標價
- **E 產業分析師觀點** → 領先指標 + 一句話建議
- **F 思考導航** → 三個延伸追問

每篇報告有 **🟢 / 🟡 / 🔴 鮮度標示**,提示哪些章節已過期。

| 指標 | 監控對象 | 資料源 |
|------|---------|--------|
| **① 漲停股** | 全市場漲停數 + 產業集中度 | TWSE/TPEx daily K |
