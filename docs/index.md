## 📊 DDMD 個股分析

用 **「大大毛多」** 框架系統化拆解個股,完整 A→F 六章節報告:

- **A 公司總覽** → 公司是誰、賣什麼、賣給誰、性感題材
- **B 大大毛多評估** → 產業 / 市佔 / 毛利率 / 成長性 四維度評分
- **C 評分 + 三句話敘事** → 0-20 分總評 + 「為何贏 / 怎麼賺 / 怎麼輸」
- **D 股價情境分析** → 悲觀 / 中立 / 樂觀 / 極度樂觀 四情境目標價
- **E 產業分析師觀點** → 領先指標 + 一句話建議
- **F 思考導航** → 三個延伸追問

每篇報告有 **🟢 / 🟡 / 🔴 鮮度標示**,提示哪些章節已過期。

## 📡 黑飛舞短線策略

「**創高雙黑站 5MA(強勢回檔)**」短線進場策略,經 **9.5 年 / 7357 筆 trigger 全市場回測**:

| 指標 | 數值 |
|------|------|
| 樣本期間 | 2016-10-31 ~ 2026-05-04(含 2018 貿易戰 / 2020 疫情 / 2022 熊市 / 2024 AI 牛市) |
| 勝率 | **32.6%** |
| 平均獲利 | +9.49% |
| 平均虧損 | -2.81% |
| **每筆期望值** | **+1.20%** |

搭配 **四段式動態停利** — Stage A 緊停損保命 / Stage B+C 用 5MA 移動 / Stage D 飆股放大。

## ⚠️ 重要免責聲明

本站所有內容**僅供研究參考,不構成投資建議**。

- 股票投資具有風險,可能導致本金損失
