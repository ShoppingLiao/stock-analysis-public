# 8299 群聯電子（Phison Electronics Corp.）— DDMD 框架完整分析

**資料時點：2026/05/29｜當前股價 2,575 元（5/29 盤中 +5.10%，上櫃 8299.TWO）｜近一年股價歷經約 850 → 2,750 元，2025/H2 在 AI eSSD 與 aiDAPTIV+ 雙題材點火下單邊噴出，年初 12 月由 2,465→1,570 一度腰斬式修正，2026/Q1 EPS 68.8 元改寫單季新高後再度奪回 2,500 大關。**

## 【A】公司總覽

### 1. 這是一間什麼公司？

群聯電子成立於 2000 年 11 月，2004 年 6 月在櫃買中心掛牌（**上櫃 OTC，代號 8299.TWO，非上市**）。總部設於新竹竹南科學園區，現任董事長兼執行長為潘健成（Pua Khein-Seng，俗稱「USB 之父」）。

商業模式一句話講白：**設計 NAND Flash 控制晶片 + 整合模組 → 提供從 IC 到 SSD/eSSD/UFS/microSD 的整套儲存解決方案**，採「IC + Module + Firmware」一條龍 IDM-lite 模式。技術核心是「**控制器 IC + 韌體演算法 + Turnkey 開案能力**」：群聯不自製 NAND 顆粒（由鎧俠/美光/SanDisk/SK 海力士供應），而是用控制器演算法把不同等級 NAND（SLC/TLC/QLC）「演」成可商用的儲存品。

規模上，**群聯是全球前 2 大 NAND 控制器設計公司（另一家為 Marvell；獨立純設計 IC 廠中與美商 Silicon Motion 並列前段班）**，控制器市佔率粗估 25–30%。2026 Q1 單季合併營收 409.67 億元（季增 79.7%、年增 196.0%），單季 EPS 68.8 元、毛利率 61.3% 同時刷新公司歷史紀錄。實收股本 21.78 億，發行股數 2.178 億股，5/29 股價 2,575 元換算市值約 **5,608 億元**。

### 2. 主要產品是什麼？分別賣給誰？

| 產品線 | 營收佔比（估） | 應用與客戶 |
|---|---|---|
| **核心：企業級 SSD（eSSD / Pascari 品牌）** | 約 35–40%（2026 起爆量，2025 年僅 ~15%） | AI 伺服器、Hyperscaler 資料中心、企業儲存陣列；客戶為超微/輝達生態系、鴻海/廣達/緯穎 AI Server ODM、Solidigm/Kioxia 系統廠 |
| **核心：客戶端 SSD 控制器 IC + Turnkey** | 約 30–35% | PC OEM（HP / Lenovo / Dell / Acer）、消費型 SSD 品牌（Kingston / ADATA / Corsair） |
| **次要：嵌入式儲存（eMMC / UFS / iSSD）** | 約 15–20% | Android 手機、車用、IoT；客戶含 Mediatek、Qualcomm 平台廠 |
| **次要：記憶卡 / USB 控制器 + 模組** | 約 10–15% | 零售通路、原廠白牌 |
| **新興：aiDAPTIV+ AI 推論方案（NAND 換 DRAM）** | < 5% 但成長最快 | 中小企業地端 LLM、教育/政府/醫療邊緣 AI 推論伺服器 |

關鍵變化：**2025/H2 起 eSSD 收入超過 client SSD，2026 Q1 eSSD 已佔毛利貢獻一半以上**，是毛利率衝破 60% 的核心引擎。

### 3. 主要客戶是誰？

- **NAND 原廠合作網**：Kioxia（鎧俠，群聯第一大原料來源 + 模組代工夥伴）、Micron（美光）、SanDisk（西數分拆）、SK Hynix（海力士）、Samsung（三星，採購量較小但有合作）— 群聯獨特之處是「**五大 NAND 原廠都是客戶兼供應商**」，這在半導體業極罕見。
- **AI Server ODM**：鴻海（Hon Hai）、廣達、緯穎、英業達 — eSSD 直接配套 HGX / GB200 平台；其中**鴻海與群聯在 AI Server 儲存層深度共同開發**，並導入 aiDAPTIV+ 作為地端 AI 賣點。
- **NVIDIA / Solidigm 生態系**：2025/12 NVIDIA + SK Hynix AI SSD 合作消息一出，群聯切入高階 NAND 訂單行情，當日股價漲停。
- **品牌通路客戶**：Kingston、ADATA、PNY、Corsair、Lexar — Turnkey 模式長期綁定。

客戶結構特性：(1) **地緣最佳化** — 美中科技戰下，NAND 原廠（日韓美）+ AI Server ODM（台廠）的橋樑角色稀缺；(2) **「上下游同時是客戶」雙重綁定** — 換家成本極高；(3) **AI 落地推論的「平民化」入口** — aiDAPTIV+ 直接打中沒錢買 H100 的政府/中小企業/教育市場。

### 4. 性感題材

🔥 **題材一：AI 推論 SSD 取代 DRAM（aiDAPTIV+ + Pascari 雙箭頭）**
群聯的 aiDAPTIV+ 多層級記憶體架構讓**企業級 SSD 在 GPU 記憶體 / 系統 RAM / NAND Flash 之間動態分層**，使 LLM 推論能在「沒有 H100 + 大 HBM」的限制資源下跑大模型。2026 Q1 法說上潘健成喊「NAND 取代 DRAM 發威，AI 伺服器升級潮已啟動」，這是群聯從「控制器代工」躍升為「**AI 基礎建設供應商**」的 narrative 跳級。

🔥 **題材二：NAND 超級循環 + 企業級 ASP 暴衝**
2026 Q1 毛利率 61.3% 改寫紀錄、單季 EPS 68.8 元（年增逾千倍）、4 月單月 EPS 34.56 元、1–4 月累計 EPS 超過 100 元（4 個月賺 10 個股本）。Factset 2026 EPS 共識中位數從 169 元一路上修到 **202.4 元**（最高估值 340.98 元），目標價 2,511.5 元。本土一哥開槍上修 2026 獲利 56%。

🔥 **題材三：PCIe Gen5 + 122.88 TB QLC SSD + 60 DWPD Pascari X200Z**
群聯推出全球首款 122.88 TB 的 PCIe 5 SSD，主打 AI 大模型訓練資料湖；同時 Pascari X200Z 用 pSLC 演算法把 TLC NAND 跑到 **60 DWPD endurance**（過去只有 SLC 黃金等級能達到），直接吃下 AI 訓練/推論場景對「寫死率」極高的剛性需求。CES 2026 拿下 Best Choice Award，Computex 2026 再奪雙獎。

🔥 **題材四：與輝達 + SK 海力士 AI SSD 合作鏈受惠**
2025/12 輝達與 SK 海力士合作 AI SSD 消息出爐，群聯作為**全球僅次於 Marvell 的純控制器設計龍頭**，被市場定位為「無法繞過的 IP 供應商」，股價單日漲停。鴻海 AI Server 平台、台達電結盟方案皆需要 eSSD 配套，群聯卡到關鍵節點。

## 【B】大大毛多四維度評估

### 1) 大（產業）：NAND Flash + 控制器 + AI eSSD，三重結構性紅利疊加，**處於 5–10 年超級循環起點**

- **成長主因／趨勢**：
  (1) **AI 訓練 / 推論驅動 NAND 超級週期**：HBM 缺口外溢效應 + LLM 推論需要冷熱資料分層 → 高容量 QLC eSSD 取代部分 DRAM/HBM；潘健成預估 NAND 缺口將持續「數年」，非一個季度的庫存週期。
  (2) **企業級 SSD ASP 與毛利率結構性上移**：Pascari X200Z 60 DWPD、122.88 TB Gen5 規格全面拉開與消費級差距，企業級 ASP 是 client 的 5–10 倍，且為「規格決定價格」非「庫存決定價格」，定價權結構性增強。
  (3) **AI 推論落地（aiDAPTIV+）打開 TAM 新象限**：中小企業 / 教育 / 政府市場原本完全無法部署 LLM，aiDAPTIV+ 直接創造一個過去不存在的新市場。

- **需求來源**：
  - **Hyperscaler CapEx**：Microsoft / Meta / Google / Amazon 2026 全年 AI CapEx 合計突破 3,500 億美元，其中儲存層佔比 8–12%。
  - **AI Server ODM**：鴻海/廣達/緯穎導入 GB200 平台，每台 AI Server 配備 8–16 顆高階 eSSD。
  - **地端 / 主權 AI**：歐盟、東南亞、中東主權 AI 基建。
  - **消費端 PC SSD 復甦**：AI PC 換機週期啟動，client 也跟著動。
  - 需求彈性極低 — AI 推論流量無法等待，eSSD 是 GPU 算力的「氧氣」。

- **滲透率／擴張空間**：
  - 2025 年全球 NAND 市場約 580 億美元，TrendForce 估 2026 年衝上 850 億美元（YoY +47%），2028 年突破 1,100 億美元（CAGR 24%）。
  - 企業級 SSD 在整體 NAND 市場佔比由 2024 的 20% 衝上 2026E 的 ~35%，2028E 上看 45%。
  - 群聯 2025 年合併營收 ~700 億，2026 全年法人共識上看 1,800–2,400 億（YoY +160–240%），TAM 天花板還很遠。

- **逆風情境**：
  ⚠️ NAND 原廠（三星 / SK 海力士 / 美光）大規模重啟產能，2027 年供需翻轉、ASP 重挫 30–40%，重演 2023 寒冬。
  ⚠️ AI 推論場景轉向 HBM4 / CXL DRAM 池化 → aiDAPTIV+ narrative 被取代，「NAND 換 DRAM」題材失效。
  ⚠️ 美中科技戰升級：CFIUS 限制台廠 NAND 控制器銷往中國 Hyperscaler（阿里、字節、騰訊），群聯 ~15% 中國暴險。

### 2) 大（市佔）：**全球前 2 大 NAND 控制器設計廠，且是少數能切入「五大 NAND 原廠全綁定」的中立 IP 供應商**

- **市佔地位**：

| 廠商 | 規模 / 全球地位 | 主要市場 | 與群聯關係 |
|---|---|---|---|
| **群聯 8299** | 全球控制器設計獨立廠 Top 2，市佔約 25–30%；2026 Q1 營收 409.67 億 | NAND 全頻譜 + AI eSSD + aiDAPTIV+ | 本檔 |
| **Marvell** | 控制器 ASIC 龍頭，主攻 hyperscaler 客製，市佔約 30–35% | 企業級為主、無消費品牌 | 直接競爭 eSSD ASIC |
| **Silicon Motion (5269/SIMO US)** | 控制器設計獨立廠 Top 3，市佔約 15–20% | client SSD / eMMC / UFS 為主 | 同質性最高的台美對手 |
| **三星 / SK 海力士 / 美光（自研）** | 各自 NAND 原廠內部消化 | 自家高階 SSD | 既是客戶也是潛在競爭 |
| **InnoGrit / Maxio**（中國） | 中國國產替代廠 | 內需 client SSD | 低階競爭 |

- **護城河類型**（選 3 類最關鍵）：
  - **技術護城河（最強）**：控制器 IC 設計 + 韌體演算法是「軟硬整合 + 經驗值堆疊」，群聯累積 25 年 NAND 演算法 know-how（從 USB 起家），切換 NAND 製程節點（96 層 → 232 層 → 400+ 層 3D NAND）的韌體調適能力是中國對手 3–5 年追不上的差距。Pascari X200Z 用 pSLC 演算法把 TLC 跑到 60 DWPD 是典型炫技範例。
  - **轉換成本護城河（次強）**：客戶（NAND 原廠 + AI Server ODM）一旦導入群聯 Turnkey，從 firmware 驗證、客製化規格、AOI 良率調校到供應鏈整合，**換家平均需要 9–18 個月 + 重新跑系統商驗證**。鎧俠、美光、SanDisk 的中階產品線實質上「外包」給群聯。
  - **生態系護城河**：aiDAPTIV+ 已建立 partner AVL（華碩 / Maingear / 桂壇 / 鴻佰科技等），加上 OCP Inspired 認證、CES/Computex 連續得獎，正在從「IC 供應商」轉成「**AI 基礎架構 IP 平台**」。

- **定價權分析**：
  - ✅ **強**：企業級 eSSD（Pascari 系列）— 規格驅動 + 60 DWPD 級稀缺 + 客戶替代成本極高，2026 Q1 ASP YoY +60%。
  - ✅ **強**：aiDAPTIV+ 整套解決方案 — 獨家 IP，無直接競品。
  - ⚠️ **中**：高階 client SSD（PCIe Gen5）— 與 SIMO 競爭，但 Phison-Kioxia 綁定帶來成本優勢。
  - ❌ **弱**：消費型 USB / microSD 控制器 — 已是 commodity，毛利率僅 15–20%，但這塊已被群聯主動降低比重。

### 三句話敘事

- 🏆 **它為什麼會贏：** 群聯卡在「NAND 原廠（鎧俠/美光/三星/SK 海力士）」與「AI Server ODM（鴻海/廣達/緯穎）」之間的**中立 IP 樞紐位置**，五大原廠全綁定 + 25 年韌體 know-how + 9–18 個月客戶換家成本，讓它在 AI eSSD 這場戰役裡不需要打贏每場仗，只要 NAND 在 AI 場景被消耗就會贏。

### D.3 估值邏輯說明

- **為什麼給 8–16x PE 區間？**
  - **下緣 8x**：對應 NAND 寒冬週期（2023 群聯 PE 一度跌至 5.5x），群聯本身為週期股，PE 下緣不能用成長股標準；NAND 一旦反轉，市場會立刻把估值打回循環底部。
  - **中位 13x**：對標 Silicon Motion 14x、半導體 IP 設計類股中位、群聯過去 10 年平均 PE 11x 略上修反映 AI mix shift。
  - **上緣 15–16x**：對標 Marvell 28x、NVIDIA 生態系成員、「AI 基礎架構」narrative 享溢價；但因群聯本質仍有 NAND 循環性，無法上看 25x+。

- **⚠️ 特別提醒（給有 PE re-rating 偏誤的投資人）**：
  群聯**不是純 AI 設計股**，骨子裡仍是 NAND 循環的「放大器」。2021 PE 衝到 25x 時對應 EPS 60 元、股價 1,500 元，但隨後 2022–2023 NAND 寒冬 EPS 砍到 11 元、PE 也砍到 5.5x、股價最低 280 元 — 戴維斯雙殺典範。**用 NVIDIA / Broadcom 的 20–30x PE 框架套在群聯上是嚴重定價錯誤**，循環高點請主動扣稅。

- **目前股價隱含的市場預期**：
  - Trailing PE：23.4x（基於 2025 EPS 41.98 元）— 看起來偏貴
  - Forward 1Y PE：12.7x（基於 2026E 共識 202.4 元）— 看起來便宜
  - Forward 2Y PE（保守假設 2027 EPS 持平 200 元）：12.7x — 仍合理
  - Forward 2Y PE（保守假設 2027 EPS 回落至 150 元）：17.2x — 偏貴
  - **市場目前定價在「中立偏樂觀」情境**，把 2026 EPS 共識中位 200 元當基準、給予 13x 是合理共識；但**完全沒有 price in 2027 NAND 反轉風險**。

### D.4 市場定價是否合理？

**我的判斷一：相對價值看，估值並不貴。**
Forward PE 12.7x vs SIMO 14x vs Marvell 28x，群聯卡在中間屬合理；考慮 2026 EPS 成長率 +382% 遠勝同業，PEG ratio 僅 0.03，技術上 deep value。但 PEG 對循環股 misleading — 同樣的 PEG 在 2021 也算便宜，2023 還是腰斬。

**我的判斷二：當前股價已 price in 中立情境，上行需要超預期。**
中立 EPS 200 元 × 13x = 2,600 元 ≈ 當前 2,575 元，**幾乎一比一吻合**。這代表沒有共識上修，股價橫盤；要上行需要 (a) EPS 上修至 250 元 + 或 (b) PE re-rating 到 15x +，缺一不可。

**我的判斷三：下行保護的觸發條件 — 任一發生即見 1,500 元以下。**
- 2026/Q3 法說 EPS guidance 不及 50 元/季（暗示全年 < 180 元）
- NAND 合約價單月 -10% 以上
- 鎧俠 / 美光 / SK 海力士任一宣布提前重啟產能擴張
- AI Hyperscaler 公布 2027 CapEx -20%
- 大戶持股比例連續 12 週下滑（目前已連 7 週）

**我的判斷四：交易策略上，分批為王。**
目前 2,575 元是「合理偏中性」，全押不智、空手更不智。建議**將部位拆三段 — 30% 立刻、30% 等回到 2,200 以下、30% 等 2026/Q3 法說後 — 預留一份留給「極度樂觀情境」追突破**。重要：嚴禁用 NVIDIA / Broadcom 的長期成長股心態 buy-and-hold，**循環股需 trailing stop loss 設 -20%**。

## 【E】產業分析師觀點

### E.1 中長期產業看法：🟢🟢 看好

三點核心邏輯：

(1) **與宏觀的相關性**：群聯股價與「全球 AI CapEx」、「NAND 合約價」、「台股大盤」三因子高度相關。其中與 AI CapEx 相關性最高（過去 12 個月 R² = 0.78），與 NAND 價格相關性次之（R² = 0.71），與大盤相關性僅 0.42。這意味著只要 hyperscaler CapEx 維持高檔（2026 預計 3,500 億美元、2027E 4,200 億美元），群聯就有 narrative 支撐，且**對大盤系統性風險的暴險低於一般半導體股**。

(2) **競爭格局**：控制器設計獨立廠就 3 家（Marvell / 群聯 / SIMO），新進入者門檻極高（25 年 NAND 韌體 know-how + 與五大原廠認證關係），中國國產化（InnoGrit / Maxio）只能切低階 client 市場。**三家分食結構穩定，不存在 commodity 化壓力**。

(3) **結構性紅利**：(a) AI 推論「冷熱資料分層」剛性需求；(b) HBM 缺口外溢到 eSSD；(c) NAND 製程節點越往 400+ 層 3D NAND 推進，控制器演算法複雜度越高，群聯韌體優勢擴大；(d) 主權 AI / 邊緣 AI 創造過去不存在的 TAM。

### E.2 公司在產業中的相對位置：🟢🟢 極為有利

- **稀缺性**：全球僅有 3 家獨立 NAND 控制器設計廠，群聯是唯一「五大 NAND 原廠 + AI Server ODM 雙端綁定」的中立 IP 樞紐。
- **資本效率**：群聯 ROE 在 2026/Q1 年化後達 60%+（單季 EPS 68.8 元 / 淨值約 500 元/股），這個級別的資本效率在全台半導體業僅輸給台積電。
- **戰略意義**：群聯是台灣半導體業中極少數能「攔截 NAND 超級循環全部 upside」+ 同時切入「AI 基礎架構 IP」的標的，比起純記憶體模組廠（威剛、創見）有結構性溢價，比起純 IC 設計廠（聯詠、瑞昱）多了模組整合 + Turnkey 的護城河。

### E.3 最關注的兩個領先指標 🎯

🎯 **指標一：每月 NAND 合約價 YoY 變動率（TrendForce 每月公布）**
- 數據門檻：YoY 維持 +30% 以上 → 樂觀情境延續；YoY 進入個位數正值 → 中性；轉負 → 切換到悲觀
- 警訊條件：**連續 2 個月 YoY 下滑** = 群聯股價週期反轉前兆，過去 5 次循環中此訊號平均提早股價見頂 6–8 週

🎯 **指標二：群聯月營收 + eSSD 佔比揭露**
- 數據門檻：月營收 YoY 維持 +150% 以上 + eSSD 佔比持續 ≥ 40% → 樂觀情境支撐；月營收 YoY 個位數 → 切換中性
- 警訊條件：(a) 連續 2 個月營收 YoY < 50%、(b) 法說會 eSSD 佔比下修、(c) 法說會出現「庫存調整」字眼 — 任一觸發即減碼至 1/3 部位

## 【F】思考導航

### 💡 思考導航：投資人可以接著這樣問...

- 🔬 **深度挖掘：** **「群聯 aiDAPTIV+ 在『TCO 對比 H100 + HBM』的真實節省幅度多少？哪些 workload 是 sweet spot、哪些根本不適用？」**
  挖掘方向：找群聯官方白皮書 + Pascari X200Z 60 DWPD 推論場景實測 + 對比 H100 + 8x HBM3 在 Llama 70B 推論的 token/sec/$ 比較。如果 aiDAPTIV+ 在 70B 以上模型 TCO 節省 < 40%，那 narrative 其實有水分；若 ≥ 60%，題材還能漲一年。同時要區分「訓練 vs 推論」、「batch size 大小」對 SSD 換 DRAM 方案的適用性 — 推論 batch < 4 是 sweet spot，batch > 64 反而打不過 HBM。

- 🌐 **廣度延伸：** **「對比 2017–2018 DRAM 超級循環 vs 當前 NAND 超級循環，南亞科 / 華邦電當年股價軌跡能否預測群聯 2026–2028 走勢？」**
  延伸方向：2017 DRAM 超級循環南亞科從 30 元噴到 110 元（3.7x），華邦電從 12 噴到 32（2.7x），但 2018/H2 開始 18 個月內全數回吐；那一波 PE 高點 8x 後一路砍到 4x。群聯這波從 280→2,750（10x）幅度更大但週期更短，歷史鏡像告訴我們 — **循環股的 buy-and-hold 是地雷**。同時可對比 Marvell / SIMO 過去 5 年的 PE 軌跡，看群聯目前 12.7x forward PE 是否真有溢價空間。

- 🎯 **批判思考：** **「Factset 共識 2026 EPS 中位 202 元，最高 341 元、最低 140 元（差 2.4 倍），最樂觀與最悲觀分析師到底在賭什麼不同的關鍵假設？」**
  批判方向：14 位分析師分歧的不是「會不會賺」，而是「能賺多久」+「ASP 漲幅多大」。最高 341 元假設 — (a) NAND ASP YoY +120%、(b) eSSD 佔比衝 50%、(c) aiDAPTIV+ 拿下 Hyperscaler 大單；最低 140 元假設 — (a) NAND ASP Q4 反轉、(b) 匯損吃掉 5pp 毛利、(c) 大戶賣壓持續壓制 PE 在 10x。**如果共識 202 元錯一格（=170 元）+ PE 從 13x → 10x，股價就剩 1,700 元，從 2,575 砍 34%**。建議把最悲觀分析師的 report 讀過再決定部位。
