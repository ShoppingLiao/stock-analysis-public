## 【A】公司總覽

### 1. 這是一間什麼公司？

- **成立／上市**：1996/06/27 創立，股票代號 **3149**，所屬產業「光電 — 玻璃加工」。
- **商業模式一句話**：把康寧、肖特、AGC 等玻璃龍頭的原板，**做奈米級的切、磨、薄、強、鍍、鑽**，再依客戶規格交付給品牌端 — 它是「玻璃加工代工廠」，不是玻璃材料廠。
- **技術核心**：3D 玻璃成型、薄化（thinning）、化學強化、CNC 精密加工、玻璃通孔（TGV）製程、AR/AG/AF 鍍膜。**奈米級加工能力**是其護城河本體 — 這是被康寧主動上門尋求合作的關鍵。
- **規模／集團地位**：國內**最大**玻璃薄化與拋光廠，鴻海集團子公司（前三大股東「鴻元 7.01% + 鴻揚 4.48% + 鴻棋 4.27% = 15.76%」均為鴻海集團法人，代表人皆為黃德才）。市值約 177–195 億（依股價水準），股本約 24.8 億。

### 2. 主要產品是什麼？分別賣給誰？

- **核心產品線一：消費性電子玻璃保護蓋（智慧型手機 / 平板 / 智慧手錶用蓋板玻璃）— 仍佔本業大宗**
  - 主要供應 Apple iPhone / iPad / Watch 系列玻璃蓋板（鴻海代工生態系內部供應鏈），2026 Q1 季營收約 4.7 億，毛利率仍為負（-4.14%），是 2025 全年負毛利率 -16.5% 的主因。
  - 應用場景：消費電子終端品牌客戶。

- **核心產品線二（轉型重點）：HDD 玻璃疊盤（伺服器 / AI Data Center 用 HAMR 熱輔助磁性儲存碟）**
  - 2026/03/20 董事會決議投資 10 億元建置量產線；月產能目標 **2026 年底 100 萬片 → 2027 Q4 達 500 萬片**。
  - 法人預估毛利率 **25%–30%**（遠高於目前負毛利的消費電子玻璃），有機會在 2027 變成主力獲利引擎。
  - 客戶：直接對應 Western Digital / Seagate / 鎧俠等 HDD 廠（為 AI 伺服器大容量儲存配套）。

- **次要 / 卡位產品線三：半導體先進封裝玻璃基板（CoWoS / CoAS / CoPoS）**
  - 康寧 2026 上半年主動上門尋求合作，正達是其指定的台灣去中化代工夥伴。
  - 量產時程：**2027 H2 才放量**，目前處驗證 / 試產階段；法人估「毛利率相當不錯」（未具名數字，市場推估 30%+）。
  - 客戶：透過康寧後端供應台積電（CoWoS / CoPoS）、Intel（Glass Core Substrate）、Samsung 電機等先進封裝路線。

- **次要 / 題材卡位產品線四：軍工 / 航太用玻璃**
  - 子公司「正達戰術科技」與美系國防承包商簽 5 年框架協議；航太級玻璃適形天線進實測。
  - 客戶：美系國防客戶（Firestorm、Aerkomm）；下半年可能有實質採購訂單。

- **次要 / 早期卡位產品線五：AI 眼鏡 / 車用玻璃**
  - AI 眼鏡為短期股價題材之一，但具體出貨進度未公開；車用玻璃逐步從消費電子轉移過去，是利基產品線。

### 3. 主要客戶是誰？

- **終端品牌**：Apple（透過鴻海代工體系供 iPhone / iPad / Watch 蓋板）、HDD 兩大廠 Western Digital / Seagate（伺服器玻璃碟盤客戶，正在驗證 / 鎖訂單中）、美系國防承包商（Firestorm、Aerkomm，已簽 5 年框架協議）。
- **產業鏈合作夥伴（最關鍵的）**：**康寧（Corning, GLW US）— 全球玻璃龍頭主動找上門**，這是公司 2026 估值大幅 re-rate 的核心轉折。康寧本身的玻璃基板 + 大猩猩玻璃在全球居絕對龍頭地位（光通訊業務銷售已與 Meta 簽近 200 億美元長約，AI 玻璃材料定價權極強），其指定正達做台灣端的「奈米加工夥伴」就是把正達拉進半導體先進封裝供應鏈。
- **客戶結構特性**：
  1. **集團綁定（鴻海生態系）**：iPhone 蓋板訂單長期受惠鴻海代工體系內部優先；半導體 + AI Server 訂單也順著鴻海雲端 + AI Server 業務外溢。
  2. **去中化稀缺性**：過去玻璃加工 70%+ 在中國（藍思、伯恩光學等），但 China + 1 趨勢下，**台灣具備奈米級玻璃加工能力的廠商屈指可數**，正達是少數能對接國際大廠的選項。
  3. **龍頭認證效應**：被康寧選為合作夥伴後，等於拿到「進入半導體先進封裝玻璃基板供應鏈的入場券」。

### 4. 性感題材（市場最關注的成長故事）

- 🔥 **題材一：康寧 × 正達 — 玻璃龍頭認證的稀缺加工夥伴**
  康寧 2026 上半年主動上門尋求合作（不是正達去爭取，是康寧自己找上門），正達搭上「半導體先進封裝玻璃基板供應鏈」順風車。康寧自身有 AI 玻璃定價權（光通訊產能近售罄、Meta 200 億美元長約），其指定的台灣代工夥伴就有題材天花板。

- 🔥 **題材二：HDD 玻璃疊盤 — AI Server 儲存升級的必然受惠**
  AI 訓練與推理對大容量、低成本儲存的需求暴增，傳統鋁碟片在 HAMR（熱輔助磁性錄寫）製程下會變形，玻璃碟片是唯一解。正達投資 10 億元建線，2026 年底月產 100 萬片 → 2027 Q4 月產 500 萬片（**3.5 年內產能擴 5 倍**），毛利率 25%–30%，是純獲利貢獻新業務。

- 🔥 **題材三：CoPoS / TGV 玻璃基板 — 台積電下世代封裝平台**
  台積電 CoPoS（Chip-on-Panel-on-Substrate）2026 在采鈺建測試線、2027 試產、2028 嘉義廠正式量產。Intel 玻璃核心基板 2026–2030 年程；Samsung 電機目標 2026–2027 量產玻璃芯基板。**全球玻璃基板市場 2026 年 ~170–186 億美元 → 2030 年 ~243–320 億美元（CAGR 6–14.5%）**，先進封裝為高毛利甜蜜點。

- 🔥 **題材四：軍工無人機 + 航太玻璃天線 — 高毛利利基切入**
  子公司正達戰術科技 + 美系國防承包商 5 年框架協議，2026 H2 可能有實質採購；航太級玻璃適形天線進實測 — 軍工 / 航太雙位數毛利率，可望挹注本業由負轉正。

## 【B】大大毛多四維度評估

### 1) 大（產業）：**從「消費電子玻璃」（成熟低毛利）整列遷移到「半導體先進封裝玻璃 + AI 儲存玻璃」（高成長高毛利）**

- **成長主因／趨勢**：
  (1) **AI 算力升級驅動先進封裝材料革命** — 矽中介層在大尺寸 Chiplet + HBM 堆疊面臨翹曲、熱應力、切割浪費等天花板，玻璃基板熱膨脹係數接近矽、平整度高、低介電損耗成為 AI/HPC 新解；
  (2) **AI Data Center 儲存需求暴增** — 訓練 / 推理產生海量資料，傳統鋁碟片無法支撐 HAMR 技術，玻璃碟片成為 HDD 容量升級唯一路徑；
  (3) **地緣政治 / China + 1 強制轉單** — 玻璃加工原本 70%+ 在中國，國際品牌客戶（Apple、康寧、台積電）要求供應鏈去中化，台灣具奈米加工能力者**屈指可數**。

- **需求來源**：
  - **AI Server / HPC（最強）**：NVIDIA H100/B200/Rubin 系列、AMD MI300/MI400、Google TPU、AWS Trainium 等先進封裝玻璃基板需求；HDD 廠（WDC、Seagate）為 AI Data Center 提供大容量儲存。
  - **半導體大廠驗證隊伍**：台積電、Intel、Samsung 電機、安靠（Amkor）皆規劃 2026–2028 試產 / 量產玻璃基板路線。
  - **需求彈性**：先進封裝玻璃基板的需求對單價的彈性**低**（被指定材料），HDD 玻璃碟對 AI Server CapEx 高敏感。

- **滲透率／擴張空間**：
  - 全球玻璃基板市場 **2026 ~170–186 億美元 → 2030/2031 ~243–320 億美元**，CAGR 6%–14.5%（不同機構口徑差異大，反映新技術不確定性）。
  - HDD 玻璃疊盤滲透率：目前 HDD 玻璃碟 vs 鋁碟比例約 10–15%，HAMR 全面採用後可望 5 年內提升至 50%+，**TAM 至少 3–5 倍**。
  - 半導體玻璃基板 2027 H2 起放量，2028–2030 才是真正放量段。

- **逆風情境**：
  - ⚠️ **量產時程延後 ≥ 6 個月**：CoPoS / Glass Core 涉及複雜驗證，台積電 / Intel 任一方延後驗證，正達 2027 放量計畫直接打折，股價情境 -30%–-40%；
  - ⚠️ **本業仍在虧錢消化新事業 CapEx**：玻璃保護蓋本業 2025 毛利率 -16.5%、Q1 2026 仍 -4.14%，HDD + 玻璃基板新線投資 10 億元，折舊壓力 2026 全年仍可能讓 EPS 維持小幅虧損；
  - ⚠️ **AI Server CapEx 放緩**：若 2027 年北美 Hyperscaler CapEx 增速從 +30% 降至 +10%，HDD 玻璃疊盤訂單就會延後，500 萬片月產能目標可能跳票。

- **目前股價隱含的市場預期**：
  - Trailing PE：無意義（虧損）。
  - Forward 1Y PE（用 2026E 接近損平估）：仍無意義。

### 市場定價是否合理？

**我的判斷一：相對價值 — 略貴但還沒到瘋狂。**
正達 forward 2Y PE ~40x，相對於欣興 22x、康寧 25x、藍思 20x，溢價 60–100%。這個溢價在「轉機股 + 4 條新產品線同步推進 + 鴻海集團背書 + 康寧合作」的故事下勉強合理，但**沒有任何犯錯空間**。

**我的判斷二：隱含 forward PE 落點偏「樂觀」。**
目前股價已隱含 2027 EPS 至少 +1.8 元的預期，但事實上：
- 2026 H2 才會看到 HDD 玻璃首批小量出貨
- 2027 H1 才會看到 CoPoS / TGV 試產良率
- 軍工 5 年框架協議落地細節未公開
任何一條晚於市場預期 6 個月，股價就要從現價向中立情境（72 元 +/- 10%）回測。

**我的判斷三：下行保護不足 — 觸發 -30% 修正的條件比想像中容易。**
最容易觸發的劇本：
1. Q2 / Q3 2026 法說 EPS 仍為負 0.3 元以上 — 市場開始質疑「轉機故事」；
2. 任一新事業（HDD / CoPoS / 軍工）公開傳出進度延後；
3. AI Server CapEx 2027 增速從 +30% 降至 +10–15%；
4. 鴻海集團法人賣超超過外資買超 1.5 倍。
任兩條同時發生，股價可在 4–8 週內從 71 → 50 元（-30%），技術面跌破月線（~52 元）後可能進一步測 4 月底起漲點 ~47 元。

**我的判斷四：當前籌碼結構偏熱、短線高風險。**
- 融資使用率 43.17%（過去 1 年新高區）
- 單日當沖率 51.44%（超過一半成交是當沖客）
- 投信 0 參與（沒有長線法人接手）
- 6/1 開盤 78.3 元跳空 +9.97% 是情緒高潮表現
**目前股價更像「題材氛圍 + 散戶 + 短線外資」共振推升，不是基本面共識**。

## 【E】產業分析師觀點

### 1. 中長期產業看法：看好 🟢🟢

3 點核心邏輯：
(1) **與宏觀的相關性**：玻璃基板需求高度綁定 AI Server / Hyperscaler CapEx — NVIDIA Rubin / AMD MI400 / Google TPU v7 / AWS Trainium 3 等 2027–2028 平台升級必經之路；HDD 玻璃疊盤綁定 AI 訓練 / 推理對長期儲存的剛性需求。AI CapEx 至少還有 3–5 年高速增長期，**產業順風期未過半**。
(2) **競爭格局**：全球能做半導體級奈米玻璃加工的廠商屈指可數 — 中國藍思、伯恩在消費電子規模碾壓但未進半導體驗證；美 SCHOTT、肖特做材料端不做加工；台灣只有正達 + 少數幾家中小廠能對接國際大廠。**寡占格局有利定價權**。
(3) **結構性紅利**：China + 1 是不可逆趨勢 — 美中科技脫鉤強制 Apple / 康寧 / 台積電 / Intel 把關鍵供應鏈搬離中國，台灣是首選替代地，**正達是「結構性受惠 + 龍頭認證」的雙重受惠者**。

### 2. 公司在產業中的相對位置：有利 🟢

相對於同業的稀缺性：
- **規模面**：消費電子玻璃市佔輸藍思、伯恩 5–10 倍 — 大宗市場不是它的戰場。
- **技術面**：奈米級加工 know-how + 鴻海集團 R&D + 康寧背書 — 半導體先進封裝供應鏈的入場券不是用錢買的，是用「驗證履歷 + 龍頭認證」換的。
- **戰略意義**：對鴻海集團而言，正達是「集團半導體後段封裝玻璃供應鏈的橋頭堡」 — 鴻海 2024–2025 強推 AI Server / GPU 模組業務，正達 HDD 玻璃 + 玻璃基板剛好補上儲存與封裝兩條鏈，集團內部資源傾斜可期。
- **資本效率**：但要警惕 — 公司過去 5 年連續虧損，ROE 為負，新事業 10+ 億元 CapEx 對淨值衝擊大；**戰略意義佳但資本效率仍未驗證**。

### 3. 最關注的兩個領先指標 🎯

- 🎯 **指標一：HDD 玻璃疊盤月產能進度 — 2026 年底是否達 100 萬片/月**
