# 🇹🇼 台股 DDMD 分析(stock-analysis-public)

公開版台股 DDMD 框架深度分析,用「大大毛多」(產業 / 市佔 / 毛利率 / 成長性)四維度系統化評分。

## 🌐 線上看

**https://shoppingliao.github.io/stock-analysis-public/**

## 📖 內容

- 個股 DDMD 完整分析報告(A~F 六章節)
- 黑飛舞策略說明 + 回測結果
- 評分標準 / 框架說明
